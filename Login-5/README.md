# Login 5  
**En este proyecto se intenta dar algunos estilos e implementar alguna funcionalidad como acceder a un json para comprobrar si el usuario está registrado, al igual que se pueda avisar cuando un formato de correo no sea válido.  
Las funcionalidades como recuperar la contraseña o registrar el usuario no son implementadas en este proyecto. Además, claro, de mejorar las prácticas.  
Los lenguajes utilizados fueron HTML, CSS y JavaScript.**
